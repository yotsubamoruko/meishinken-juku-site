Space

自然光とハーブの気配がある、ちょうどいい広さ。

テラス、窓際、カウンター、ボックス席。写真が映えるだけでなく、ひとり時間にも数人の集まりにも使いやすい構成を想定しています。

Terrace / Window

夕方の光が入るテラス席は、ペット連れや軽い待ち合わせにぴったり。写真もきれいに撮れます。

Counter / Table

一人でも入りやすいカウンターと、2〜4人で使いやすいテーブル席をバランスよく配置します。

Seats

店内24席 + テラス6席。小さすぎず、でもうるさくなりすぎないサイズ感。

Comfort

Wi-Fi / 電源 / ベビーカーOK / ペット可。都会の中で使いやすい設計です。

Style

白壁、オーク、セージグリーン。気取らないのに、きちんとおしゃれな空気感。

FAQ

よくある質問を、先に見せる。

参考サイトと同じく、利用前の不安を先に解消するのが大事です。予約導線の前に、短く読みやすいFAQを置きます。

予約は必要ですか？

ランチは予約なしでも入りやすい想定ですが、ディナーや土日の利用は予約優先にしています。

子ども連れでも大丈夫？

ベビーカーでの入店がしやすい導線を想定し、テーブル幅もゆったりめにしています。

支払い方法は？

キャッシュレスのみ。クレジットカード、交通系IC、QR決済に対応する設定です。

Access / Hours

名古屋駅前で、ふらっと寄れるイタリアン。

架空の店舗設定として、名古屋駅西口から徒歩5分の立地に。ランチ、カフェ、ディナーの時間帯を分けて、自然に使い分けられる導線にしています。
